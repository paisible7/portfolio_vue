## Guide pour le Portfolio

### Champs obligatoires :
1. **Titre** : `# Nom du Projet` (première ligne)
2. **Image** : `<img src="..." />` ou `![Banner](./path)`
3. **Description** : Texte après `## Description` avec `>`
4. **Type** : Dans `## Liens` → `- **Type:** web` ou `- **Type:** mobile`
5. **Demo** : Dans `## Liens` → `- **Demo:** https://...`
6. **Tags** : Dans `## Stack Technique`

### Valeurs pour Type :
- `web` → Icône lien externe
- `mobile` → Icône téléchargement
