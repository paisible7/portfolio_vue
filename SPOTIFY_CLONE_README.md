# Spotify Clone

<img src="https://upload.wikimedia.org/wikipedia/commons/1/19/Spotify_logo_without_text.svg" alt="Banner" width="100%" />

## Description
> Clone fonctionnel de Spotify développé avec Vue.js 3. Reproduit l'interface principale, la navigation musicale, la gestion de playlists et l'expérience de lecture audio, optimisé pour desktop et mobile.

## Liens
- **Type:** web
- **Demo:** https://spotify.paisible.tech

## Stack Technique
* **Framework:** Vue.js 3
* **Langage:** TypeScript
* **Style:** Tailwind CSS
* **Backend:** Supabase
* **Outils:** Vite, Pinia, Lucide Icons

## Fonctionnalités
- **Lecture Audio** : Lecteur audio complet avec contrôles (play, pause, suivant, précédent)
- **Playlists** : Création et gestion de playlists personnalisées
- **Recherche** : Recherche de titres, artistes et albums
- **Responsive Design** : Interface optimisée pour desktop et mobile
- **Mode Sombre** : Thème sombre natif comme Spotify

## Installation

1. **Cloner le projet**
   ```bash
   git clone https://github.com/paisible7/SPOTIFY_CLONE.git
   ```

2. **Installer les dépendances**
   ```bash
   cd SPOTIFY_CLONE
   npm install
   ```

3. **Configurer Supabase**
   - Créez un projet sur [Supabase](https://supabase.com)
   - Copiez vos clés dans un fichier `.env`

4. **Lancer en mode développement**
   ```bash
   npm run dev
   ```
